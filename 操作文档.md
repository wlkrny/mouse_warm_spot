## 操作流程

### 1. 打开视频
菜单栏 → 文件 → 打开视频 (Ctrl+O)

### 2. 绘制两个圈

| 圈 | 颜色 | 作用 | 如何画 |
|---|---|---|---|
| **ROI Core** | 绿实线 | 暖点铜片精确范围，判断是否压暖点 | 鼠标拖拽绘制 |
| **ROI Count** | 蓝虚线 | 两只鼠可能出现的范围，用于计数 | 绘制内圈后再拖拽一次 |

工具栏按钮可在编辑内圈/外圈间切换。

### 3. 标记参考帧 (0/1/2)

工具栏有 3 个按钮：

| 按钮 | 作用 |
|---|---|
| `[0] 标记背景` | 跳到空场帧，点击标记暖点无遮挡画面 |
| `[1] 标记1只` | 跳到只有1只鼠的帧，可多次点击追加样本 |
| `[2] 标记2只` | 跳到有2只鼠的帧，可多次点击追加样本 |

- **左键** = 追加样本
- **右键** = 撤回/清空
- **不要求按顺序**，[0] 未标时其他标记暂存，标了 [0] 后自动重算

### 4. 自动检测

点击工具栏 **"自动检测全视频"** → 进度条显示 → 完成后事件列表出现在左下角。

### 5. 审核事件

| 操作 | 快捷键 |
|---|---|
| 点击事件 | 视频跳转到该片段并循环播放 |
| 选择小鼠身份 | `1` `2` `3` `4` (多选 toggle) |
| 确认数量=1 | `Ctrl+1` |
| 确认数量=2 | `Ctrl+2` |
| 保存并下一个 | `Enter` |
| 标记误检 | `X` |
| 清空选择 | `C` |
| 播放/暂停 | `Space` |
| 单帧前进/后退 | `←` `→` |
| 快进/快退 | `D` `A` |
| 上/下一个事件 | `J` `L` |
| 刷新计数 | `R` |

事件列表颜色：
- 🟡 黄 = 待审核
- 🟢 绿 = 已确认
- 🔴 红 = 误检
- 🟠 橙 = 低置信需审核

### 6. 颜色识别 (耳标辅助)

点击工具栏 **"颜色识别"** → 后台扫描当前片段 → 弹窗显示识别到的颜色 → 输入对应鼠号(1-4) → 自动标记。

### 7. 导出

| 菜单 | 快捷键 | 格式 |
|---|---|---|
| 文件 → 导出CSV | Ctrl+E | 30字段 CSV (UTF-8 BOM) |
| 文件 → 导出Markdown统计表 | - | 完整时间线 + 小鼠汇总 + 校准摘要 |

## 核心算法摘要

### 占据检测 (ROI Core 绿圈内)
- 粗筛：每10帧检测 ROI Count 蓝圈区域
- 精查：触发后回溯2秒逐帧检测
- 状态机判断进入/离开，cap上限=2只

### 数量估计 (ROI Count 蓝圈内)
- 前景 = 强暗色(V<55) OR (暗候选(V<95) AND 背景差异>25)
- 连通区 → 过滤(面积/长宽比/接触ROI Core≥20px) → 碎片合并 → blob计数
- 面积计数：面积比 <1.7→1只, 1.7-2.7→2只
- 单blob最多判2只低置信，绝不判3/4

### 耳标颜色
- 搜索 ROI Count 环形区域(排除ROI Core)
- 三轮帧扫描
- 条件: V>80 S>30 H非暖色, 3-500px, 靠近小鼠身体≤20px
